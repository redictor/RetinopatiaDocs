Инструкция по эксплуатации приложения RetinopatiaApp

1. Запуск приложения

После запуска приложения на экране отображается окно авторизации.

В данном окне пользователь может:

  • Войти в существующий аккаунт
  • Перейти к созданию нового аккаунта

Для авторизации необходимо:

  1. Ввести логин аккаунта
  2. Ввести пароль
  3. Нажать кнопку «Войти»

2. Регистрация нового пользователя

Если аккаунт отсутствует необходимо нажать кнопку «Создайте новый».

  1. Введите логин
  2. Придумайте пароль
  3. Повторите пароль
  4. Нажмите кнопку «Зарегистрироваться»

3. Главный экран приложения

После авторизации открывается главный экран приложения.

На главной странице отображается:

  • Приветствие пользователя
  • Краткая сводка статистики обучения
  • Последняя активность

Кнопки быстрого доступа:

  • Начать обучение
  • Открыть статистику
  • Настройки

4. Переход в режим обучения

Для начала тренировки необходимо выбрать пункт «Обучение» в боковом меню.

5. Этап 1 — запуск тренировки

На первом этапе необходимо нажать кнопку «Начать тренировку».

После этого приложение автоматически загружает изображение глазного дна из встроенного набора данных.

6. Этап 2 — выбор стадии и разметка изображения

После загрузки изображения необходимо:

  1. Выбрать предполагаемую стадию диабетической ретинопатии
  2. С помощью кисти отметить подозрительные области
  3. При необходимости использовать ластик

7. Подтверждение разметки

После завершения разметки необходимо нажать кнопку «Подтвердить выбранные данные».

Это действие фиксирует выбор пользователя и переводит тренировку на следующий этап.

8. Проверка результата с помощью ИИ

Для запуска анализа необходимо нажать кнопку «Проверить результат».

ИИ анализирует изображение, определяет стадию патологии и строит тепловую карту области внимания.

9. Просмотр результата обучения

После завершения анализа отображается результат.

Пользователь может увидеть:

  • Стадию, определённую ИИ
  • Стадию, выбранную пользователем
  • Оценку совпадения результатов
  • Итоговый балл от 1 до 5
  • Сравнение области внимания пользователя и ИИ

10. Статистика обучения

Во вкладке «Статистика» отображается прогресс пользователя.

  • Общее количество тренировок
  • Средняя оценка
  • Соответствие области внимания
  • График динамики результатов

11. Настройки аккаунта

Пользователь может:

  • Изменить пароль
  • Выйти из аккаунта
  • Удалить аккаунт

12. Завершение работы

Для завершения работы можно:

  • Выйти из аккаунта
  • Закрыть окно приложения
